XSLT Blender Portal

Least power, greatest effect

This project serves as home to several demonstration applications using standards-based XML/XSLT technologies, on a commodity stack running in your browser. A demonstration may be here in order to show a possibility; to model a concept (supported by an open, auditable, replicable code base); to provide a useful utility application; or all of these together.

On this basis, this site and projects presented here are also designed to illustrate and to document XML/XSLT-based applications, especially with respect to issues of data and system security when operating on an open stack and an open network.

By and large and in accordance with this objective, many demonstrations are designed to read inputs provided by the user and process them securely on the user's behalf, without exposure.

For further documentation of this open-source project, with its aims and objectives, see the project repository. For details not just overview (especially regarding risk assessment and risk management), see the project wiki.

Test the runtime

How is this a test

"View All Projects" loads an XML file, listing current and prospective (proposed) projects.

This also exercises the transformation runtime: if you can see projects listed (not a blank or error), other demonstrations can be expected to work as well.

If the directory fails to load, try a recent browser, supporting XSLT 1.0 with the DOM API.
