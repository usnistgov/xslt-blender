XSLT Blender Portal

Least power, greatest effect

For a description of this project, its aims and objectives, see the project repository.

Loading the directory reads an XML file in the repository listing current and prospective (proposed) projects. A link to an active project opens it in another browser window.

If the directory fails to load, try a browser supporting XSLT 1.0 with the DOM API.
