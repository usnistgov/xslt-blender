HTML Reducer

Drop your XHTML onto the page. Your browser loads it and converts it to a plain, simple, flat, Markdown-compatible HTML.

Upcoming features:

  • Textbox for literal HTML
  • Three renderings: in layout; XHTML (OSCAL-ready) source; Markdown
  • Save As and copy-to-clipboard options for XHTML and Markdown
  • Find a way to support handling non-wf HTML (such as Tidy)?

Limitations

Currently this can only process XHTML or other XML (well-formed) inputs. If your HTML is not already XML you can Tidy it first.

HTML Reducer is an XSLT Blender project using XSLT 1.0 capabilities built into your browser, with no external dependencies.
